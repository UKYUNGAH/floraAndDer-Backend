# 1주차 코드 리뷰

공통 사항
1. 모든 경로에서 .DS_Store 파일을 지우시고 commit + push를 해주세요. 브랜치를 하나 만들어서 하셔도 되고 아니면 main브랜치에서 하셔도 될 것 같습니다.
2. README.md에 있는 내용 수정해주세요. 제 이름이 그대로 나와서 나중에 문제가 될 수도 있을 것 같습니다.
3. mongoose의 스키마의 필드명들이 어떤 경우는 camel 어떤 경우는 snake로 되어있는데 "통일은" 되어야할 것 같아요.
4. 코드 스타일이 정리가 안되어 있습니다. 가령 인덴테이션이 안되어있는 경우 등이 있어요. Prettier를 사용해서 한 번에 정리를 해줍시다. 이건 오피스아워 때 함께 시도해봐도 좋을 것 같네요.
5. 에러 메시지가 어디는 한국어이고 어디는 영어인데 언어 통일이 필요해보입니다.
6. .gitignore를 제가 업데이트 해두었습니다. 해당 파일로 교체해주시면 보다 많은 불필요한 파일들이 git에서 제외가 될 겁니다.


개별 사항
- 나머지 리뷰 내용은 주석으로 달았습니다. 찾기 편하게 // C: 라는 형식으로 작성되어있으니 찾아보시면 될 것 같아요.

1주차 정말 고생 많으셨습니다. 모두 서버 어플리케이션 개발이 처음이실텐데 예상보다 더 잘하셨습니다. 
자신감을 가지시면 좋을 것 같습니다. 나머지 한 주 힘내시고 언제든지 질문이 있으시면 연락주세요.
감사합니다.

한인호 드림.
